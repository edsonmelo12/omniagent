# Test Prompts - Creative Director

Use these prompts to validate whether the skill is producing distinct creative directions instead of repeating the same layout logic.

## Test 1: Value perception batch

Prompt:

> We need a new client batch about perceived value. The pieces should feel related, but each one must use a different visual logic. Avoid repeating the same center composition or the same headline treatment. Give me a creative brief for 4 assets.

Expected signal:

- different directions per asset
- one proof-led item
- one manifesto item
- one backstage item
- one closing item

## Test 2: Clarity and education batch

Prompt:

> Plan a clear educational batch for a registered client. The goal is to simplify the message and make the pieces easy to scan on mobile. I want 4 assets with different compositions, not just different copy.

Expected signal:

- editorial or educational direction
- strong hierarchy
- modular structure for at least one item
- clear anti-repetition warnings

## Test 3: Proof and trust batch

Prompt:

> Build a batch for a client focused on trust and proof. The assets should show evidence, craft, and credibility, but not all in the same visual pattern. Recommend 3 art directions and explain which one should lead the batch.

Expected signal:

- proof-led direction
- backstage direction
- one alternative with comparison or closing
- clear recommendation and rationale

## Test 4: Closing campaign batch

Prompt:

> We are closing the campaign and need the final client assets to feel conclusive and memorable. The last piece should not look like the first piece with a different line of text. Give me a batch plan with distinct visual roles.

Expected signal:

- closing or manifesto direction
- sequence planning
- final asset feels like a conclusion
- no duplicated structure across the batch

## Test 5: Repetition check

Prompt:

> Compare this new client batch with the previous rendered assets and tell me if the new direction is too similar. If it is, propose a different family and show how the composition should change.

Expected signal:

- explicit similarity analysis
- warning if the batch is too close to the previous one
- different focal logic proposed
- practical recommendation for a new direction
