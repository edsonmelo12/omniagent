# Art Directions - Client-Agnostic

## Purpose

Este arquivo descreve famílias visuais que a skill pode escolher ao planejar uma peça para qualquer cliente cadastrado.

A regra principal é simples:
cada item do lote precisa poder cair em uma direção diferente, para evitar repetição estrutural.

## Direction 1: Editorial

### When to use

- quando a peça precisa de clareza
- quando o texto tem função explicativa
- quando a leitura precisa ser rápida e limpa
- quando a autoridade da mensagem importa

### Visual traits

- hierarquia forte
- respiro generoso
- composição limpa
- blocos bem separados
- alto foco em legibilidade

### Best for

- posts educativos
- posts com tese
- comunicação de valor
- explicação de oferta

### Avoid

- excesso de ornamento
- muitos blocos competindo
- composição muito centralizada o tempo inteiro
- excesso de efeitos

## Direction 2: Proof-Led

### When to use

- quando a peça precisa provar algo
- quando o objetivo é aumentar confiança
- quando há detalhe, processo ou resultado para mostrar

### Visual traits

- prova em destaque
- texto curto e incisivo
- composição com âncora visual clara
- leitura de valor sem esforço

### Best for

- bastidores
- depoimentos
- resultados
- validação de oferta
- argumento de credibilidade

### Avoid

- texto longo demais
- abstração sem evidência
- layout genérico de “card bonito”
- prova escondida no fundo

## Direction 3: Backstage

### When to use

- quando o foco é processo
- quando a peça precisa parecer humana
- quando o valor vem do fazer, não só do resultado

### Visual traits

- mais textura
- camadas visuais
- sensação de rotina
- recortes, detalhes, materiais
- composição menos rígida

### Best for

- bastidores
- craft
- produção
- suporte
- rotina operacional

### Avoid

- acabamento excessivamente polido
- visual frio demais
- falta de detalhe humano
- muita simetria

## Direction 4: Manifesto

### When to use

- quando a peça precisa marcar postura
- quando a campanha quer convicção
- quando a frase principal precisa dominar

### Visual traits

- headline grande
- contraste alto
- estrutura simples
- pouco ruído
- gesto visual forte

### Best for

- tese da campanha
- frase de posicionamento
- encerramento de lote
- peça de assinatura

### Avoid

- encher de informação
- competir headline com elementos secundários
- layout fraco ou tímido
- excesso de elementos decorativos

## Direction 5: Comparison

### When to use

- quando a peça mostra contraste
- quando existe antes/depois
- quando o objetivo é provar transformação

### Visual traits

- divisão clara
- dois lados bem definidos
- foco na diferença
- leitura imediata da mudança

### Best for

- comparação de oferta
- problemas vs solução
- erro vs acerto
- antes vs depois

### Avoid

- divisão confusa
- demasiadas informações por lado
- ambiguidade visual
- falta de contraste real

## Direction 6: Educational

### When to use

- quando a peça ensina algo
- quando há sequência lógica
- quando a mensagem precisa ser escaneável

### Visual traits

- módulos curtos
- progressão clara
- estruturas em blocos
- bom uso de títulos e subtítulos

### Best for

- carrossel
- posts de ensino
- conteúdo de clareza
- posts com “3 pontos” ou “passo a passo”

### Avoid

- densidade excessiva
- falta de progressão
- blocos longos sem respiro
- layout que parece apostila

## Direction 7: Closing

### When to use

- quando a peça encerra um lote
- quando a mensagem precisa ficar na cabeça
- quando o objetivo é fechar com clareza e memorização

### Visual traits

- foco único
- síntese
- composição mais calma
- leitura final forte

### Best for

- tese final
- recado de marca
- call to action mais limpo
- encerramento de ciclo

### Avoid

- múltiplas mensagens
- excesso de camadas
- fechamento fraco
- peça sem sensação de conclusão

## Direction 8: Commercial Direct

### When to use

- quando a peça precisa vender mais diretamente
- quando a oferta precisa aparecer com nitidez
- quando a decisão já está próxima

### Visual traits

- clareza comercial
- CTA explícito
- foco na oferta
- baixa ambiguidade

### Best for

- venda direta
- campanha final
- post de conversão
- peça com chamada concreta

### Avoid

- linguagem abstrata demais
- estética excessivamente conceitual
- baixa legibilidade de CTA
- excesso de história antes do ponto principal

## Direction Selection Rules

A skill deve escolher a direção com base em:

- objetivo da peça
- formato
- posição da peça no lote
- histórico visual recente
- nível de prova disponível
- intensidade da mensagem

## Direction Alternation Guide

Evite repetir a mesma direção em sequência.

Exemplo de alternância boa:

- editorial
- proof-led
- backstage
- manifesto

Exemplo de alternância ruim:

- editorial
- editorial
- editorial
- editorial

## Recommended Pairings

### When the batch is about clarity

Use:

- editorial
- educational
- closing

### When the batch is about trust

Use:

- proof-led
- backstage
- commercial direct

### When the batch is about positioning

Use:

- manifesto
- editorial
- closing

### When the batch is about transformation

Use:

- comparison
- proof-led
- educational

## Direction Definition Standard

For every direction chosen, the skill should specify:

- family name
- mood
- composition logic
- focal element
- text density
- CTA style
- what to avoid

## Final Rule

A direction is only valid if it changes how the piece reads.

If it only changes color, it is not a new direction.
If it only changes text, it is not a new direction.
If it changes the visual logic, then it is.
