# Examples - Creative Director

## Example 1: Batch about value perception

### Context

- Client: registered client
- Batch: batch-01
- Goal: increase perceived value
- Channel: Instagram Reel and Feed
- Risk: pieces becoming too similar

### Recommended direction split

- Item 1: manifesto
- Item 2: backstage
- Item 3: comparison
- Item 4: proof-led

### Why this works

- The first piece opens with posture.
- The second shows process.
- The third demonstrates contrast.
- The fourth ends with credibility.

### Expected briefing outcome

```md
# Creative Brief

## Batch Context
- Client: registered client
- Batch: batch-01
- Channel: Instagram Reel
- Objective: increase perceived value

## Recommended Direction
- Family: Proof-led editorial
- Mood: confident, clean, purposeful
- Composition: one dominant focal area with supporting proof
- Focal Point: product detail and value cue

## Alternatives
1. Backstage with process detail
2. Comparison with before/after logic
3. Manifesto with bold thesis

## Per-Item Notes
- Item 1: headline-led opening
- Item 2: process-led composition
- Item 3: contrast-led structure
- Item 4: proof-led closing

## Anti-Repetition Warnings
- Do not repeat the same centered layout
- Do not use the same CTA position in every item
- Do not keep the same text weight across the batch

## Prompt Guidance
- Keep the frame clean
- Use a different focal logic for each item
- Let the proof read instantly on mobile

## Approval Checklist
- Clear focal point
- Different from previous item
- Good mobile legibility
- Batch variety preserved
```

## Example 2: Batch about clarity

### Context

- Client: registered client
- Batch: batch-02
- Goal: improve clarity and reduce confusion
- Channel: Instagram Feed
- Risk: educational cards all looking the same

### Recommended direction split

- Item 1: editorial
- Item 2: educational
- Item 3: comparison
- Item 4: closing

### Why this works

- Editorial creates authority.
- Educational gives structure.
- Comparison shows a change in reading.
- Closing ends the sequence cleanly.

### Expected briefing outcome

```md
# Creative Brief

## Batch Context
- Client: registered client
- Batch: batch-02
- Channel: Instagram Feed
- Objective: reduce confusion and create clarity

## Recommended Direction
- Family: Editorial
- Mood: clean, direct, structured
- Composition: text-led with strong spacing and one supporting visual anchor
- Focal Point: key message of clarity

## Alternatives
1. Educational with modular blocks
2. Comparison with clear before/after
3. Closing with a concise thesis

## Per-Item Notes
- Item 1: strong opening and title hierarchy
- Item 2: modular explanation
- Item 3: contrast and transformation
- Item 4: final concise synthesis

## Anti-Repetition Warnings
- Avoid the same layout in every card
- Avoid repeating the same size headline
- Avoid repeating the same visual anchor

## Prompt Guidance
- Vary the structure across the batch
- Keep the reading order obvious
- Use contrast and spacing to separate ideas

## Approval Checklist
- Readable on mobile
- Different focal logic
- Fits the client tone
- Adds diversity to the set
```

## Example 3: Batch about support and closing

### Context

- Client: registered client
- Batch: batch-03
- Goal: close the campaign with trust and routine
- Channel: Instagram Feed
- Risk: calm pieces becoming visually flat

### Recommended direction split

- Item 1: backstage
- Item 2: proof-led
- Item 3: educational
- Item 4: manifesto or closing

### Why this works

- Backstage adds human texture.
- Proof-led adds credibility.
- Educational adds utility.
- Closing adds memory and conclusion.

### Expected briefing outcome

```md
# Creative Brief

## Batch Context
- Client: registered client
- Batch: batch-03
- Channel: Instagram Feed
- Objective: close with trust and clarity

## Recommended Direction
- Family: Backstage
- Mood: warm, useful, stable
- Composition: process-led with visible craft and a clean focal area
- Focal Point: support and routine

## Alternatives
1. Proof-led support
2. Educational routine map
3. Manifesto closing statement

## Per-Item Notes
- Item 1: show process
- Item 2: show trust
- Item 3: show simplicity
- Item 4: show thesis

## Anti-Repetition Warnings
- Avoid using the same centered card structure
- Avoid using the same palette across every asset
- Avoid finishing every piece with the same CTA

## Prompt Guidance
- Make each piece feel like a different visual sentence
- Keep the batch coherent but not monotone
- Let the final piece feel conclusive

## Approval Checklist
- Clear purpose
- Distinct structure
- Strong enough variety
- Consistent with the client identity
```
