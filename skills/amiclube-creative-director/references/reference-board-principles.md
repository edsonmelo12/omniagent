# Reference Board Principles

Source: `templates/criativos/exemplos.png`

This board is a useful creative reference for any client because it shows a wide range of social post treatments.

## Use it for

- composition variety
- hierarchy decisions
- typography intensity
- photo versus graphic balance
- whitespace and density control
- editorial rhythm across a batch

## Extracted principles

1. One dominant idea per post.
2. Avoid relying on a single centered template.
3. Alternate between photo-led and type-led posts.
4. Use contrast deliberately, not as decoration.
5. Keep supporting blocks subordinate to the main hook.
6. Let whitespace raise the perceived quality.
7. Vary the visual rhythm across the cycle.
8. Make the final piece feel finished, not like a sketch.

## Risks

- some examples are less polished than others
- a few layouts expose their skeleton too much
- some typography treatments are strong while others are weak

## Rule for the client

Use the board as a direction source, not as a template source.
