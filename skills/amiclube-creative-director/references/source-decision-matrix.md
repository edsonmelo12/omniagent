# Source Decision Matrix

This matrix defines the default free-source class by post type.
When external imagery is used, prefer assets that show craft, tools, workspace, hands, materials, or other artisan signals before neutral imagery.

| Post Type | Default Source Class | Use External Source? | Notes |
|---|---|---|---|
| Editorial card | No external source | No | Typography-led.
| Checklist card | No external source | No | Graphic-led.
| Comparison card | No external source | No | Use internal composition.
| Recap card | No external source | No | Summary-first.
| Manifesto carousel | No external source | No | Keep it typographic.
| Educational carousel | No external source | Usually no | Only source if clarity improves.
| Product shot + proof card | Unsplash / Pexels or CC0 / Public Domain | Yes | Use when context strengthens the value statement.
| Process shot / reel | Unsplash / Pexels or CC0 / Public Domain | Yes | Use for workspace or craft context.
| Proof reel | Unsplash / Pexels or CC0 / Public Domain | Yes | Use for evidence-driven context.
| Reassurance carousel | No external source or CC0 / Public Domain | Maybe | Use only if a calm context image helps.
| Value card | Unsplash / Pexels or CC0 / Public Domain | Yes | Use when proof needs visible support.
| Text-first post | No external source | No | Do not force imagery.

## Rule

Choose the least risky source class that still improves the piece.
If an image is used, it should carry at least one signal from the artisan niche instead of reading as filler.
