# LinkedIn Carousel

This skill adapts the shared social visual system to LinkedIn carousels.

- `../social-visual-system/SKILL.md`
- `../social-visual-system/references/visual-foundation.md`
- `../social-visual-system/references/component-system.md`

## When to use

Use this skill when the post should feel like:

- a concise document
- a framework
- a diagnosis
- a strong professional opinion
- a case summary

## Format rules

- Prefer 4 to 8 slides.
- Use more breathing room than Instagram.
- Allow more text, but keep hierarchy obvious.
- Avoid fake app chrome, swipe arrows, and playful UI cues.
- Prefer restrained backgrounds, stronger grids, and calmer rhythm.

## Content rhythm

Suggested sequence:

1. thesis
2. problem or context
3. framework or contrast
4. evidence or explanation
5. action or recommendation
6. CTA or closing takeaway

## CTA style

LinkedIn CTAs should feel professional:

- save this
- share with your team
- comment with your scenario
- follow for more frameworks

## Output

Return a LinkedIn-ready carousel brief with:

- slide sequence
- typography direction
- visual restraint notes
- CTA style
- export recommendation
