# Validation Checklist

Use this checklist when evaluating whether a client creative direction is ready for render.

## Validation Gates

### 1. Clarity and value
- One dominant idea only
- Hook readable on mobile
- Value signal obvious fast

### 2. Educational structure
- Sequence is easy to follow
- Blocks are distinct
- Density fits the format

### 3. Proof and trust
- Proof is visible
- Proof is specific
- Proof supports the thesis

### 4. Closing strength
- The piece feels conclusive
- The final frame is memorable
- The ending matches the batch goal

### 5. Anti-repetition
- Focal logic differs from the previous asset
- Composition changes, not just color or copy
- The batch gains real variation

## Rejection Rule

If the piece fails any gate, revise the direction before render.

If three or more pieces in the same batch share the same composition logic, revise the batch before approval.
