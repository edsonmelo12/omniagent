# Source Selection Guide

This guide helps the skill decide whether to use an external free source and which source class to prefer.
When possible, choose imagery that shows niche signals like hands, tools, materials, workspace, team, environment, or product context before neutral lifestyle imagery.

## Source Classes

1. No external source
2. CC0 / Public Domain support image
3. Unsplash / Pexels support image
4. Wikimedia Commons reference
5. Pixabay fallback

## Decision rules

- Use no external source when the piece is type-led or graphic-led.
- Use CC0 / Public Domain when reuse freedom matters most.
- Use Unsplash / Pexels when the piece needs high-quality free photography.
- Use Wikimedia Commons only when the exact file license is verified.
- Use Pixabay as a low-risk fallback.

## What to record

- source class
- source name
- URL
- license type
- attribution requirement
- intended use

## Final rule

If the asset does not improve the piece, do not source it.
If possible, the source should read as part of the artisan world instead of generic stock.
