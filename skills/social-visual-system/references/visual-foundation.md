# Visual Foundation

Derive a compact six-token system from the primary brand color:

- `BRAND_PRIMARY`
- `BRAND_LIGHT`
- `BRAND_DARK`
- `LIGHT_BG`
- `LIGHT_BORDER`
- `DARK_BG`

## Rules

- `LIGHT_BG` should be tinted, never flat pure white.
- `DARK_BG` should be near-black with subtle brand influence.
- `LIGHT_BORDER` should create separation without visible heaviness.
- Use one heading font and one body font unless there is a clear reason not to.

## Typography pairings

- Editorial / premium: `Playfair Display` + `DM Sans`
- Modern / clean: `Plus Jakarta Sans` + `Plus Jakarta Sans`
- Warm / approachable: `Lora` + `Nunito Sans`
- Technical / sharp: `Space Grotesk` + `Space Grotesk`
- Bold / expressive: `Fraunces` + `Outfit`
- Classic / trustworthy: `Libre Baskerville` + `Work Sans`
- Rounded / friendly: `Bricolage Grotesque` + `Bricolage Grotesque`

## Density rules

- Mobile-first formats need fewer words per frame.
- Faster-consumption formats need one message per frame.
- Professional formats can carry more text, but hierarchy must remain obvious.
