# Brand Intake

Collect these inputs before designing:

- brand name
- handle or profile name
- primary brand color
- tone of voice
- language
- logo or fallback mark
- profile photo if relevant
- product shots, screenshots, or support images
- topic or post objective
- target platform and format

If the user does not provide all fields, infer only what is safe from the
existing client context. Do not invent missing brand assets.

## Intake rules

- Prefer one strong primary color over vague palette descriptions.
- If a website is available, derive visual cues from it but keep the result
  simplified for social use.
- If the post is part of a batch, check the recent outputs first to avoid
  repetition.
