# Format Selection

Use the content goal to choose the format:

- `instagram-carousel`: depth, education, errors, frameworks, step-by-step
- `linkedin-carousel`: thesis, diagnosis, opinion, framework, case summary
- `stories-sequence`: quick rhythm, interaction, urgency, micro-proof, reply CTA
- `reels-sequence`: short-form discovery, timed scenes, replayable hooks, motion-friendly proof
- `facebook-post`: Facebook-native static feed, link preview, blog teaser, announcement, proof post
- `social-single-post`: non-Facebook one strong idea, teaser, cover, announcement, or static feed asset
- `pinterest-pin`: evergreen utility, searchable ideas, save-worthy tips

## Selection rules

- If the idea needs progression, choose a multi-frame format.
- If the idea wins by speed and clarity, choose a single-frame or story format.
- If the platform rewards search and saving, bias toward utility and scannability.
- If the platform is professional, reduce ornament and increase information clarity.
