# Review And Export

Never jump straight from concept to export without a review pass.

## Review flow

1. generate preview first
2. inspect hierarchy, spacing, and readability
3. inspect layout fit and visible-frame safety
4. check whether the format feels native to the platform
5. confirm CTA clarity
6. export only after approval

## Platform-specific dimensions

- Instagram carousel feed: `1080x1350` (4:5 portrait)
- Instagram stories: `1080x1920` (9:16)
- Facebook feed landscape: `1200x630` (1.91:1)
- Facebook feed square: `1080x1080` (1:1)
- LinkedIn: `1200x627` (1.93:1)
- Pinterest: vertical ratio optimized for discovery

## Reject conditions

- unaccented pt-BR copy
- fake Instagram chrome on non-Instagram formats
- fake Instagram preview frame on Facebook/LinkedIn exports
- unreadable text over images
- content cut below the visible frame
- layouts that only fit by excessive compression
- overpacked slides
- repetitive layouts across a batch
- Facebook export using wrong dimensions (must be 1200×630 or 1080×1080)
