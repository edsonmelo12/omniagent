# AmiClube Search Hardening Rules

To ensure Niche Fidelity and avoid generic/tech-heavy imagery (headsets, joysticks, etc.), all public image searches for AmiClube must follow these constraints.

## 1. Mandatory Inclusion Terms
Always include at least 2 of these terms in the query:
- `amigurumi`
- `crochet`
- `handmade yarn`
- `knitted toy`
- `wool craft`
- `artisan workshop`
- `soft toy handmade`

## 2. Mandatory Exclusion Terms (Negative Search)
Always exclude these terms to avoid tech/lifestyle noise:
- `-tech`
- `-electronics`
- `-gaming`
- `-headset`
- `-joystick`
- `-computer`
- `-office` (unless qualified by 'artisan' or 'craft')
- `-plastic`
- `-mass-produced`

## 3. Style Qualifiers
To maintain the "Premium Editorial" look:
- `high contrast`
- `macro texture`
- `warm lighting`
- `aesthetic craft`
- `minimalist artisan`

## 4. Validation Protocol (The "Human-Eye" Check)
Before using a link, the agent must check the image description for:
- **Material Check:** Is it yarn/wool? (If plastic or metal, REJECT).
- **Context Check:** Is it an amigurumi? (If a generic doll or tech gear, REJECT).
- **Background Check:** Is it an artisan/home environment? (If a corporate office or gaming room, REJECT).
