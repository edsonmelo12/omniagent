# Pinterest Pin

This skill adapts the shared visual system to Pinterest-native discovery posts.

## Read first

- `../social-visual-system/SKILL.md`
- `../social-visual-system/references/visual-foundation.md`
- `../social-visual-system/references/component-system.md`
- `../social-visual-system/references/review-and-export.md`

## When to use

Use this skill for:

- evergreen tips
- checklists
- practical guides
- searchable inspiration
- save-oriented educational content

## Format rules

- Prefer strong vertical composition.
- Lead with a search-friendly title.
- Make the benefit obvious in the first glance.
- Favor scannability over narrative progression.
- Use utility-first copy, not vague brand slogans.

## Structure

Good pin structure:

- clear title
- immediate benefit
- supporting visual or example
- subtle brand anchor

## Avoid

- Instagram-style chrome
- excessive CTA buttons
- overly abstract wording

## Output

Return:

- pin concept
- title structure
- utility promise
- supporting visual treatment
- export recommendation
