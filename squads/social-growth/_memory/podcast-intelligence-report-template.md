# Podcast Intelligence Report Template

## Purpose

Store a client-agnostic strategic extraction from a podcast or YouTube episode.
Use this template for any client after the transcript has been extracted.

## Required Sections

### Context Scope
- source name
- source type
- video or episode ID
- published date
- analyzed date
- client-aligned scope
- objective of the analysis

### Transcript Status
- status
- language
- available languages
- character count
- sentence count
- extraction reason or limitation

### Tese Central
- one-sentence summary of the episode's core claim

### Strategy Name
- primary strategy name
- naming type (descriptive / proof-based / platform-based / behavior-based)
- guest name(s), when available
- short source description
- secondary strategies, if any

### Estratégias Identificadas
- strategy label
- score
- number of evidences
- evidence snippets

### Provenance
- source name
- video or episode ID
- source date
- interviewed guest(s)
- source notes

### Mecanismos
- how the strategy produces results

### Táticas Mencionadas
- actionable tactics explicitly mentioned in the episode

### Aplicabilidade do Cliente-Alvo
- high / medium / low
- what should be applied
- what should be discarded
- client context notes

### Decisão Sugerida
- `scale`
- `optimize`
- `retest`
- `pause`
- `archive`

### Confiança
- high
- medium
- low

### Evidências Citáveis
- direct quotes or short evidence fragments

### Próxima Ação
- what the squad should do with this insight

### Blocking Questions
- only include if a missing input truly blocks routing or final synthesis
- otherwise, keep questions out of the report and state the working assumption

## Usage Rule

Always keep the extraction neutral first, then map it to the target client.
Never assume the source episode is automatically relevant to the current client.
Prefer assumptions over clarification loops when the evidence is sufficient to continue.
When a strategy is named, keep the source provenance attached so the squad can reuse the pattern without losing traceability.
