## Usage Rule

- o nome da estrategia deve sempre ser acompanhado de provenance
- uma estrategia pode ter varias fontes, mas cada entrada precisa registrar a origem exata
- se uma nova analise confirmar o mesmo mecanismo, atualizar a entrada sem perder historico
- se o nome ficar vago demais, simplificar para o mecanismo principal
