# Content Routing Rules

Define qual agente é responsável por criar conteúdo baseado na origem e destino.

## Routing Matrix

| Source Type | Destination | Agent | Notes |
|-------------|-------------|-------|-------|
| Estratégia/Plano Editorial (sem blog) | Instagram | **Creator** | Conteúdo social nativo original |
| Estratégia/Plano Editorial (sem blog) | Facebook | **Creator** | Conteúdo social nativo original |
| Estratégia/Plano Editorial (sem blog) | LinkedIn | **Creator** | Conteúdo social nativo original |
| Estratégia/Plano Editorial (sem blog) | TikTok | **Creator** | Roteiro/referência para Reels |
| Blog post finalizado | Instagram | **Content Repurposer** | Derivação do blog para social |
| Blog post finalizado | Facebook | **Content Repurposer** | Derivação do blog para post, Stories ou Reels |
| Blog post finalizado | LinkedIn | **Content Repurposer** | Derivação do blog para social |
| Blog post finalized | TikTok | **Content Repurposer** | Derivação do blog para social |
| Blog post finalized | Pinterest | **Content Repurposer** | Derivação do blog para social |
| Post social existente (Instagram) | LinkedIn (cross-post) | **Content Repurposer** | Adaptação entre canais |
| Post social existente (Instagram) | Facebook (cross-post) | **Content Repurposer** | Adaptação entre canais |
| Post social existente (LinkedIn) | Instagram (cross-post) | **Content Repurposer** | Adaptação entre canais |
| Post social existente (Instagram) | Stories | **Content Repurposer** | Extensão de conteúdo |

## Regras de Fronteira

### Creator
- Cria conteúdo **a partir de estratégia e plano editorial**
- Quando a fonte é blog, deixar a derivação social para o Content Repurposer
- Não faz cross-posting entre canais sociais diferentes

### Content Repurposer
- Transforma conteúdo existente (blog ou social) em novos formatos
- Trabalha com: blog→social e social→social (cross-channel)
- Quando não há trabalho a fazer (conteúdo já nativo do canal), indicar "no work needed"

## Escalação

Se houver dúvida sobre qual agente deve atuar, escalar para o **Strategist** para decisão.
