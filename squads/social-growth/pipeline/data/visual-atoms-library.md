# Visual Atoms Library (Universal)

Este documento define os componentes básicos (Átomos) que compõem o sistema visual do squad. Cada átomo tem um ID único que deve ser usado nos briefings de Direção Visual.

## 1. Atoms: Grids & Layout (Vertical 9:16)
Estes átomos definem a ocupação do espaço em Reels e Stories.

- **ATOM-GRID-V-01 (Full Density):** Topo (Contexto), Centro (Tese), Base (CTA). Preenchimento > 80%.
- **ATOM-GRID-V-02 (Focused Center):** Centro (Grande Título/Objeto), Base (CTA). Foco em impacto visual central.
- **ATOM-GRID-V-03 (List/Table):** Centro ocupado por grid de dados ou lista densa (5+ itens).

## 2. Atoms: Composition (Square 1:1)
Para Carrosséis e Posts estáticos.

- **ATOM-COMP-S-01 (Hero Impact):** Imagem total de fundo + Overlay + Título central.
- **ATOM-COMP-S-02 (Split 50/50):** Metade imagem (esquerda/direita), metade texto sólido.
- **ATOM-COMP-S-03 (Minimalist):** Fundo sólido (Átomo de Textura) + Tipografia centralizada.

## 3. Atoms: Effects & Overlays
- **ATOM-FX-BLUR-01:** Background Blur sutil (20px) para profundidade.
- **ATOM-FX-BLUR-02:** Background Blur forte (50px) para foco total no texto.
- **ATOM-FX-GRAIN-01:** Textura de ruído fino (3%) para aspecto "Film/Organic".
- **ATOM-FX-GLOW-01:** Brilho suave nas extremidades para foco central.

## 4. Atoms: UI Elements
- **ATOM-UI-CTA-01:** Botão sólido com cor de acento.
- **ATOM-UI-CTA-02:** Texto sublinhado com ícone de ação (ex: ➔).
- **ATOM-UI-BADGE-01:** Selo de categoria/série no canto superior.
