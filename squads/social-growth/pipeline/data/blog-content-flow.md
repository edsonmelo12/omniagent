# Blog Content Flow

Canonical order for long-form cycles:

1. Approve the blog architecture before drafting.
2. Create the blog draft from the approved architecture.
3. Optimize the draft for discovery.
4. Repurpose the approved blog into native social assets.
5. Assemble the content production package.
6. Review before scheduling.

## Image Research Selector

- Ask: `Deseja fazer a pesquisa das imagens agora?`
- `Sim` -> collect source notes and build the image bank for the blog/article path.
- `Não` -> keep the cycle textual and defer image work until it is actually needed.

## Rule

If image research is active, preserve the source rationale and image bank artifacts. If it is deferred, do not force those artifacts early.
