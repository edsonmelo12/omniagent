# YouTube Intelligence Brief Quickstart

## Use When

Use this when a video/podcast analysis needs to feed a new strategy cycle.

## Fast Path

1. Read the consolidated brief template.
2. Read the client adaptation template.
3. Fill the client snapshot.
4. Select the relevant strategy.
5. Translate the strategy into the client's language.
6. Remove irrelevant angles.
7. Define channel, format, proof, CTA and signal.
8. Pass the adapted brief to Step 02.

## Fixed

- conversation first
- retention as primary signal
- format as function
- proof and impact over volume
- strategy before calendar

## Variable

- niche vocabulary
- objection
- proof type
- strongest channel
- CTA priority
- KPI priority

## Output

- adapted client brief
- strategy key(s)
- decision suggestion
- validation signal

## Rule

If it still sounds generic, adapt again before using it in strategy.
