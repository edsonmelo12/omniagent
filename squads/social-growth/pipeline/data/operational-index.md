## Usage Rule

Read the files in this order when working a new client:
1. Database-backed client and product records
2. Client Intake Guide
3. Research Record
4. Research and Proposal Route
5. Open Source Social Collection Flow
6. Researcher Open Source Prompt
7. Optional Social Sales Proposal Module, when needed
8. Content Plan
9. Blog Brief
10. GEO & AI Discoverability Audit, when the site or entity is still ambiguous
11. Blog Architecture
12. Blog Draft
13. Discovery Optimization Guide
14. Discovery Optimized Blog
15. Blog Post
16. Content Repurpose
17. Content Production Package
18. Content Review
19. Schedule Plan
20. Monitoring files
