## Adaptation Checklist

1. Select the relevant strategy from the consolidated brief.
2. Translate the strategy into the client's language and offer.
3. Remove any angle that does not fit the client context.
4. Choose the channel function for the client.
5. Define the format function for the client.
6. Choose the validation signal for the client.
7. Define the CTA for the client.

## Fixed vs Variable

### Fixed

- conversation first
- retention as primary signal
- format as function
- proof and impact over volume
- strategy before calendar

### Variable

- niche vocabulary
- objection
- proof type
- strongest channel
- CTA priority
- KPI priority

## Risk Check

- Does this still sound like the client?
- Does the message stay specific enough?
- Does the format serve a real function?
- Is the CTA aligned to the funnel stage?

## Output Rule

If the answer to any risk check is no, adapt again before using it in strategy.
