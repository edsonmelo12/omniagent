# Blog Family Ledger Template

## Purpose

Track blog-family usage so neighboring articles do not collapse into the same skeleton.

## Fields

- client
- run id
- publish date
- topic
- family
- thesis angle
- opening shape
- proof placement
- nuance placement
- CTA
- visual thesis
- source notes
- repetition guardrail

## Usage Rule

Add one entry per article before final drafting.
If the next nearby post repeats the same family, the architect must justify it explicitly.
