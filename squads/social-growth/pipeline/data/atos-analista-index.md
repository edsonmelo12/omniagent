# Atos Analista - Index

## Purpose

This file is the entry point for the `Atos Analista` documentation set.
`Atos Analista` is the analytical intelligence layer that sits on top of the `social-growth` production system.

Use this index when the goal is to understand:
- the product role of `Atos Analista`
- how it differs from the production squad
- which data it consumes
- how it depends on `Otiniel Observa` for multi-client evidence intake
- how it interprets evidence
- which deliverables it must produce
