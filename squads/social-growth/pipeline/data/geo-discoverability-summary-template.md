# GEO / AI Discoverability Summary Template

## Purpose

Store the site-level GEO diagnostic before strategy and content scaling continue.
This summary is the required input layer when the site or brand entity is still ambiguous.

## Required Sections

### Context Scope
- scope type
- scope value
- scope basis
- objective
- primary audience
- product or service frame

### Canonical Positioning
- company name
- canonical page
- summary statement
- offer description
- audience description

### Entity Clarity
- naming consistency
- offer consistency
- service consistency
- audience consistency
- location or coverage consistency

### Answerability
- primary answers
- buyer questions resolved
- unanswered questions
- FAQ opportunities

### Evidence Inventory
- proof pages
- case studies
- testimonials
- metrics
- visible sources

### Structured Data
- schema present
- schema missing
- schema recommendations

### AI Readiness
- geo score
- geo verdict
- retrieval friendliness
- sourceability
- freshness
- confidence

### Risk Flags
- ambiguity
- contradiction
- thin proof
- schema mismatch
- outdated claims

### Priority Fix Order
- first fix
- second fix
- third fix

### Notes for Content Layer
- what the blog can safely assume
- what must be repaired upstream first

## Usage Rule

The research and strategy steps should read this summary before deciding whether to scale article production.
