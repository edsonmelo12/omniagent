# Reference Extraction - Creative Examples Board

Source: `templates/criativos/exemplos.png`

## What the board is good for

This board is useful as a reference because it shows:

- a wide range of post compositions
- different levels of density and whitespace
- typography-led posts and photo-led posts
- contrast between dark, light and colorful treatments
- editorial, promotional and collage-like approaches

It should be used as a reference for quality and variety, not as a direct template.

## Strong professional cues

### 1. Clear hierarchy
- Strong headlines appear fast.
- Good examples separate title, support and proof.
- The eye lands on one obvious focal point.

### 2. Composition variety
- The board does not rely on a single center layout.
- Some posts are left-heavy, some are split, some are stacked.
- This creates a more professional and less templated feel.

### 3. Real visual material
- Many examples use real photos, screenshots or product/context images.
- The best pieces do not rely only on generic decorative shapes.

### 4. Controlled boldness
- Some examples are loud, but they still keep structure.
- Strong color, heavy typography and contrast are used intentionally.
- The board shows that a post can be expressive without becoming messy.

### 5. Editorial rhythm
- Posts alternate between dense and airy compositions.
- Some pieces feel like covers, some like proof cards, some like campaign posters.
- This rhythm is what makes the board useful for professional work.

## Common weaknesses to avoid copying

### 1. Inconsistent polish
- The board mixes very polished pieces with more average ones.
- We should extract principles, not adopt every example equally.

### 2. Overloaded layouts
- Some examples use too many competing blocks.
- If the hierarchy weakens, simplify it.

### 3. Template leakage
- A few examples reveal their layout skeleton too clearly.
- We should avoid making the underlying grid too obvious in the final post.

### 4. Mixed quality of typography
- Some pieces are excellent because of type.
- Others feel weaker because the typography is less controlled.
- Typography should be tightened before production.

## Practical rules to extract

1. Use one dominant idea per post.
2. Prefer clear focal points over decorative symmetry.
3. Alternate between photo-led and type-led pieces.
4. Use strong contrast when the hook needs speed.
5. Use whitespace to increase perceived quality.
6. Keep supporting elements subordinate to the main message.
7. Do not repeat the same structure across the batch.
8. Make the post feel like a finished asset, not a layout exercise.
9. When external imagery is used, log the source class and source search note before production.

## How to apply this to the client

Use the board to guide:

- how much variety a cycle should have
- how much whitespace to allow
- when to use photos versus graphic blocks
- how bold the title treatment can be
- how to avoid the "template" look

Do not use it to:

- copy exact grids
- copy exact text placements
- copy exact color combinations without brand logic
- copy the mixed-quality examples that are less polished

## Summary

This board is a good reference for production ambition.
It shows that professional social posts can be:

- structured
- varied
- bold
- readable
- campaign-driven

The main lesson is to keep the structure visible enough to read, but hidden enough to feel designed.
