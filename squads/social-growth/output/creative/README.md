## Usage Rule

Start from the client creative profile, then open the specific document for the current step.
For image-led posts, prefer a suggestive background image, a controlled overlay, and clear typography aligned to the client palette before adding extra layout structure.
For external images, follow the source-decision matrix and log the source search note before render.
For blog posts, apply the same source hierarchy as social, but only use an image when it improves clarity, proof, scanability or editorial authority.
