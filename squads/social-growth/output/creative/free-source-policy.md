# Free Source Policy

Purpose: define a safe, low-cost sourcing path for visuals used by the social growth system.

## Scope

This policy applies to:

- support images
- background scenes
- proof/context visuals
- texture and composition references
- editorial illustrations or objects used in generated art direction

It does not replace creative direction. It only defines how to source visual material safely.

## Priority Order

### 1. CC0 and Public Domain
Use first when available.

Best for:

- maximum freedom
- lowest attribution burden
- system-wide reuse

### 2. Unsplash and Pexels
Use when the project needs high-quality free photography and the license is compatible with the intended use.

Best for:

- lifestyle support imagery
- environment photos
- generic textures or real-world scenes

### 3. Wikimedia Commons
Use when the specific file license is verified and compatible.

Best for:

- historical images
- educational references
- niche subject matter

Important:

- license must be checked per file
- do not assume all Commons files are the same

### 4. Pixabay
Use as a secondary free source when the asset fits the need and the license summary is acceptable.

Best for:

- generic stock-style visuals
- simple supporting imagery

## What to Record

For every sourced file, record:

- source class
- source name
- direct URL
- license type
- attribution requirement
- date used
- intended use
- source search note
- selection rationale

## Approval Checklist

Before using a free source asset:

- the license is clear
- the asset fits the intended use
- the asset does not carry hidden brand risk
- the asset does not look too generic for the final piece
- the asset is stored with source metadata

## Rejection Rules

Do not use the asset if:

- the license is unclear
- the file needs attribution that the workflow cannot support
- the image is too generic to elevate the final post
- the source cannot be documented
- the source is paid stock and a free/public alternative would satisfy the brief

## Final Rule

Free does not mean unverified.
Use the source only if the license is understood and documented.
Paid sources are not the default path for this system; free/public sources are.
