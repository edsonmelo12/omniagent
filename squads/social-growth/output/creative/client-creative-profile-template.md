## Decision Order

The creative skill should read these signals in this order:

1. niche
2. creative profile
3. post objective
4. post idea
5. channel and format
6. composition
7. typography
8. CTA
9. export and finishing

The niche should define the first visual angle.
The post idea should define the second visual angle.
The profile should keep the style coherent across the batch.
