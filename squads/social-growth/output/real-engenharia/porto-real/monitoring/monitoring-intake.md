# Monitoring Intake

## Inputs

- weekly post metrics
- platform insights
- comments and direct messages
- notes from sales or marketing
- any link click or visit data available

## What to Record

- post with the highest reach
- post with the highest saves
- post with the highest comments
- post with the most profile visits
- post with the best completion rate
- common objections
- recurring questions

## Reading Rule

- If a metric moves, check whether it changed intent or just visibility.
- If the audience asks the same question twice, turn it into a content topic.
- If the same post wins across cycles, keep it in rotation.
