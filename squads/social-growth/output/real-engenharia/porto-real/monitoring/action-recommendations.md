# Action Recommendations

## If Reach Grows

- keep the practical hook style;
- repeat the strongest comparison angle;
- turn the best post into a carousel sequel.

## If Engagement Grows But Intent Stays Low

- tighten the CTA;
- connect the post more directly to the buyer's decision moment;
- add more direct benefit language.

## If Video Retention Drops

- shorten the intro;
- start with the main benefit earlier;
- use faster cuts and fewer explanations.

## If Direct Messages Increase

- respond with a clear project-specific next step;
- save the main objections for the next cycle;
- build a FAQ post around the most common questions.

## If One Theme Wins Repeatedly

- turn it into a recurring series;
- create one post for each buyer question it raises;
- keep the theme in the next cycle until performance flattens.
