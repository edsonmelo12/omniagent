# Content Production Package

Content production package for Real Engenharia's first approved cycle.

## Tone Chosen

Sofisticado, confiante e claro. A marca precisa reforcar valor, projeto e decisao segura sem soar fria ou excessivamente publicitaria.

## Core Angle

Cada empreendimento resolve uma necessidade diferente. A comunicação deve ajudar o comprador a identificar qual projeto combina com o seu momento, do blog discovery ao repurpose.

## Instagram Feed

### Theme
Qual empreendimento da Real Engenharia combina com o seu momento?

### Structure
1. Capa: "Nem todo imóvel atende ao mesmo perfil. E isso é uma vantagem."
2. Slide 2: Real Towers para desejo e exclusividade.
3. Slide 3: Real Liberty para quem quer pronto para morar.
4. Slide 4: Real Paradise para quem valoriza obra concluída.
5. Slide 5: Porto Real para quem busca praticidade e custo-benefício.
6. Slide 6: CTA para salvar e compartilhar.

### Copy by Slide

- Slide 1: Nem todo imóvel atende ao mesmo perfil. E isso é uma vantagem.
- Slide 2: Real Towers: para quem busca exclusividade, frente para a baía e alto padrão.
- Slide 3: Real Liberty: para quem quer morar com conveniência e decisão mais rápida.
- Slide 4: Real Paradise: para quem valoriza entrega concluída e mais segurança.
- Slide 5: Porto Real: para quem quer praticidade, funcionalidade e um custo-benefício mais acessível.
- Slide 6: Salve este post para comparar os empreendimentos com mais clareza.

### CTA

Salve este carrossel e envie para quem está comparando imóveis em Belém.

## Instagram Reels

### Hook
Qual empreendimento faz mais sentido para o seu momento?

### Script
- Cena 1: mostrar a marca e o portfólio.
- Cena 2: destacar Real Towers em uma frase.
- Cena 3: destacar Real Liberty em uma frase.
- Cena 4: destacar Real Paradise em uma frase.
- Cena 5: destacar Porto Real em uma frase.
- Cena 6: fechar com a ideia de que o melhor imóvel é o que encaixa na necessidade do comprador.

### CTA

Comente qual empreendimento você quer conhecer melhor.

## Instagram Stories

### Sequence

- Frame 1: "Você está buscando luxo, praticidade, segurança ou entrada mais acessível?"
- Frame 2: enquete com 4 opções.
- Frame 3: resposta curta explicando o empreendimento mais votado.
- Frame 4: foto ou vídeo de apoio do projeto.
- Frame 5: CTA para responder no direct com o nome do empreendimento.

### CTA

Responda a enquete e diga qual projeto você quer ver em detalhes.

## Facebook Post

### Theme
Por que a Real Engenharia organiza o portfólio por empreendimento

### Copy

Na Real Engenharia, cada empreendimento tem um público e uma proposta de valor próprios.

Isso ajuda o comprador a entender rapidamente o que cada projeto entrega:
- Real Towers: desejo, exclusividade e alto padrão;
- Real Liberty: conveniência e pronto para morar;
- Real Paradise: confiança e obra concluída;
- Porto Real: praticidade e custo-benefício.

Quando a comunicação é segmentada, a decisão fica mais clara.
E quando a decisão fica clara, o interesse cresce com mais qualidade.

Se quiser, podemos apresentar cada projeto com mais detalhes.

## Notes for Review

- The package uses the portfolio logic instead of blending all projects into one message.
- Each platform gets a different role, but the same strategic angle.
- The CTA is focused on comparison and project interest.
