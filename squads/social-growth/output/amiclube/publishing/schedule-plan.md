## Legenda de Status

### Display (schedule-plan.md)
- `scheduled (ID)` = post criado no WordPress, aguardando data
- `preview_ready` = preview pronto para publicar
- `future` = pendente de agendamento
- `published` = publicado com sucesso

### Canônico Operacional (social-publish-queue.json / worker)
- `scheduled` = aguardando janela de publicação
- `blocked` = bloqueado por validação (caption, link, etc.)
- `published` = publicado via live API
- `failed` = tentativa falhou
- `ready_to_publish` = pronto para execução manual
