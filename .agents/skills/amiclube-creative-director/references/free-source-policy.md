# Free Source Policy

This document defines a safe sourcing path for visuals used by the creative system.

## Priority Order

1. CC0 / Public Domain
2. Unsplash / Pexels
3. Wikimedia Commons with per-file license check
4. Pixabay

## What to record

- source name
- direct URL
- license type
- attribution requirement
- date used
- intended use

## Approval checklist

- license is clear
- asset matches the use case
- no hidden brand risk
- source can be documented
- asset is not overly generic for the final piece

## Rejection rules

- unclear license
- unsupported attribution requirement
- impossible to document
- too generic for the final post

## Final rule

Free assets are only allowed when the license is verified and the file is documented.
