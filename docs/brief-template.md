# Template de Briefing Criativo

Use este template antes de acionar o gerador. Preencha todas as seções e compartilhe com os agentes + designer. A ideia é garantir propósito, tom e diferencial visuais claros.

## 1. Identidade
- **Cliente / marca:**  
- **Campanha / cápsula:**  
- **Objetivo (1 frase):** o que essa peça deve fazer na cabeça da audiência?
- **CTA final:** (ex.: “Salvar e revisar depois”, “Agendar call”, “Ver prova”)

## 2. Público & contexto
- **Público principal:** quem deve reconhecer a peça em menos de 3s?
- **Contexto de consumo:** feed, story, TikTok? momento de dia?
- **Ritmo desejado:** cadência, linguagem (irônico, direto, técnico, sensual…)

## 3. Tese / Mensagem
- **Gancho (o incômodo):**  
- **Desenvolvimento (aprofundamento):**  
- **Fecho (prova + CTA):**  
- **Referências de copy:** (insira links ou trechos)

## 4. Estética
- **Tom:** (ex.: editorial noir, brutalista, tech-luxury)
- **Palette sugerida:** (cores principais, acentos)  
- **Texturas / formas chave:** (ex.: “linha diagonal + glow rosa”, “camadas translúcidas”)  
- **Tipografia alvo:** (ex.: “display geométrica + corpo humanista”)
- **Diferenciação:** “Sem logo, será reconhecido por...”

## 5. Operação
- **Agentes envolvidos:** copywriter, visual, revisor, designer, stakeholder. Liste nomes/roles.
- **Feedback esperado:** contraste, densidade de texto, CTA.
- **Deadline:**  
- **Observações:** restrições, referenciais obrigatórios, assets necessários.

## 6. Após geração
- Liste o que revisar (macro, micro, CTA, assinatura do agente) e como registrar (issue no board? comentário no preview?).
